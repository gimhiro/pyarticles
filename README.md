# pyarticle
---

- モジュールのインストール
  ```
  pip install -r requirements.txt
  ```
- 次を実行する
  ```
  python article.py
  ```
- 実行後, `env/.env` ファイルが作成されるので以下を記述
  ```
  user_id=[qiita user_id]
  qiita_token=[qiita api token]
  slack_xoxb_token=[slack xoxb token]
  slack_channel=[slack channel name]
  ```
- 再度実行
